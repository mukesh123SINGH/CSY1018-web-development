Site Report

HTML means Hypertext markup Language, this language is mostly used for creating the website. It is most popular and widely used language in the world. It was first developed in 1991 by Berners Lee but there is lot of bugs into the language so it was again lunched in 1999. And widely used into the world. Till now HTML updated five times with different version, new version was launched into 2012 A.D. and this version is still used in the world. HTML helps the user to make the framework of website.

CSS means Cascading Style Sheets. This is widely used to decorate the HTML file. HTML is nothing without CSS because it helps give the lookup of the site. HTML provides the structural framework while CSS provide decorative things to the HTML file. Decorative material like color, text style, margin, padding and so on. It is very useful for decorate the framework designed by the user suing HTML. It is the backbone of the website. without its website looks very unsatisfying.
